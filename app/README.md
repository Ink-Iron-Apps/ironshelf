# Ironshelf App

Flutter client for the Ironshelf server.

This directory is an **M0 scaffold** — a placeholder `main.dart` and a planned `pubspec.yaml`. The real app is built in milestone M4 (see `../docs/ROADMAP.md`).

When starting M4, run `flutter create` against this directory with organization `com.inknironapps` (applicationId `com.inknironapps.ironshelf`) to generate the platform projects, then build the UI: server connection (with a custom-headers field for Cloudflare Access), Author → Series → Book browsing, sort controls, book detail with Calibre custom columns, an EPUB reader with progress sync, and the standard Ink & Iron Apps settings screens.
